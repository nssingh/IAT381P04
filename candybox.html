Designing Mockup Company Website

I was tasked to create a website and an identity for a mockup local candy shop that wants an online prescence and online store. The goals of the project were to create sketches, wireframes, static mockups and then finally code the website with HTML5 and CSS.

Sketches

I drew over 40+ sketches for 5 different possible website mockups for three different pages and for desktop and mobile viewing. The idea was to get a feel of the site layout, without committing to anything.

Wireframes

I eliminated 2 design schemes through my sketches and focused on 3 specific website designs. I created over 40+ wireframes for these 3 specific layouts. This allowed me to conduct user testing to see if the website was easy to access and navigate for information.

Mockups

These are the two final designs I decided to go with that would best represent the type of candy shop that would appeal in Vancouver's downtown core.

HTML Website

The final website combined aspects from both mockups to make a site that looked elegant, but had all the necessary options and layout for users to grasp information quickly. HTML and CSS were used to create all the pages.
